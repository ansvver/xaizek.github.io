This is a very simple thing, but I just can't make myself remember it. So here is how it's done:

  1. Tag some of messages (e.g. by using t, T, Alt-T and Ctrl-T shortcuts)
  2. Press ; (which is bound to the tag-prefix function by default) followed by any other command you want to apply to the selected items.
